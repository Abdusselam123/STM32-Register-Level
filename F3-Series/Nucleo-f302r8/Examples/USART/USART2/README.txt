
STM32 USART2 Bare-Metal Driver
This example configures USART2 on the Nucleo-F302R8 using direct register access.

⚙️ Features
Bare-metal USART2 initialization
GPIO alternate function setup (PA2 - TX, PA3 - RX)
Baud rate configuration

▶️ Usage
USART2_Init();

🧠 Highlights
RCC clock control
GPIO AF7 configuration
USART registers: CR1, BRR

📌 Note
This project is intended for learning low-level STM32 peripheral control.
